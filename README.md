## Setup
- Add `nuxt-mqtt-module` dependency using yarn or npm to your project
- Add `nuxt-mqtt-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-mqtt-module',

    // With options
    ['nuxt-mqtt-module', { /* module options */ }],
 ]
}
```
